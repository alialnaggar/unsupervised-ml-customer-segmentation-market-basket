# 🧠 Unsupervised ML: Customer Segmentation & Market Basket Analysis

---

## 📌 Project Overview

This project demonstrates **unsupervised machine learning** for customer analytics. It combines **customer segmentation** with **market basket analysis** to extract actionable insights from retail transaction data.

The goal is to answer two questions:
- **Who are the customers?** → using behavioral clustering  
- **What products are bought together?** → using association rules  

This approach helps businesses optimize marketing, sales, and inventory strategies.

---

## 🎯 Objectives

- Engineer meaningful **customer-level features**  
- Segment customers using **unsupervised ML techniques**  
- Discover product associations using **Market Basket Analysis**  
- Generate actionable **business insights**  

---

## 📊 Dataset Description

The dataset includes:
- Customer IDs  
- Transaction IDs  
- Product names and IDs  
- Quantities and total spending  
- Transaction dates  

---

## 🛠️ Methodology

### 🔹 Feature Engineering (Customer Behavior)
- **RFM & Basket Features** → Recency, Frequency, Monetary, Avg Basket Size, Total Items, Product Variety  
- **Seasonality** → Holiday shopping behavior (Q4 analysis)  
- **Occasion Profiling** → Keyword-based categories: Baby, Party, Christmas, Decor  

### 🔹 Market Basket Analysis 🛒
- Transactions grouped into baskets  
- Encoded products into a binary matrix  
- Generated **association rules** using support, confidence, and lift  

---

## 📈 Output

- **Engineered customer features** → ready for clustering  
- **Product association rules** → actionable insights for cross-selling  
- Enables segmentation by **behavior, seasonality, and intent**
